備註:您目前查看的是主版面,收錄的產品是打開即用,和子版面中的產品類型不同。

**1. 為什麼有這個列表**
身為開發者,其實蠻好奇其他人在做什麼業餘專案(不管目的是營利/興趣/試試看),
所以特意建了這個庫。歡迎各位台灣開發者把自己的專案加進來~ 發 Pull Request 或 Issue 即可 <br/>
(入選標準:開發者是台灣人或台灣團隊;主版面必須是網站或 App,不能是開發者工具或論壇型網站)

**寫好介紹語很重要**:❌ `AI 影片生成工具` → ✅ `無需剪輯經驗,3 步驟生成直式短影片`<br/>
👉 [如何寫好一句話產品介紹](./CONTRIBUTING.md)
