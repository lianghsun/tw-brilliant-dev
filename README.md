## 台灣獨立開發者專案列表

收錄台灣獨立開發者打造的作品——網站、App、遊戲與開源工具。

### 子版面

- [工程師版面](./pages/README-Programmer-Edition.md)：給工程師用的開源專案與命令列工具
- [遊戲版面](./pages/README-Game.md)：台灣獨立遊戲作品

您目前所在的主版面，收錄的是一般人打開就能用的網站與 App；需要寫程式或下指令才能使用的，請到工程師版面。

### 關於這份清單

台灣一直有許多默默把 side project 做成正式產品的開發者，但這些作品散落在 App Store、Steam、GitHub 與各個社群之間，缺少一個能一次看見全貌的地方。這份清單想做的就是這件事——把台灣獨立開發者的作品聚在一起，讓好作品被看見，也讓想投入獨立開發的人知道前面有誰走過、走到哪裡。

**收錄標準**

1. 開發者是台灣人，或以台灣為根據地的小型團隊
2. 獨立開發的作品（個人專案或 side project），而非公司主力產品
3. 主版面限「打開即用」的網站或 App；開源工具與遊戲請投稿到對應子版面

**專案狀態**

| 開發中 | 已上線 | 已關閉或缺乏維護 |
|--------|--------|--------|
| :clock8: | :white_check_mark: | :x: |

👉 想讓自己的作品出現在這裡？請看[投稿指南](./CONTRIBUTING.md)，發 Pull Request 或 Issue 都可以。

### 關於本清單的整理方式

- 這份清單由 [Claude Code](https://claude.com/claude-code) 協助蒐集、查證與整理。每筆條目都經過連結有效性與開發者背景的查證，但仍可能有疏漏——若發現錯誤（連結失效、開發者資訊有誤、不符收錄標準），請[開 Issue](https://github.com/lianghsun/awesome-taiwan-project/issues) 告訴我們。
- 歡迎台灣的開發者自薦作品！發 Pull Request 或 Issue 皆可，格式請參考[投稿指南](./CONTRIBUTING.md)。
- 清單形式受 [1c7/chinese-independent-developer](https://github.com/1c7/chinese-independent-developer) 啟發，內容為獨立收錄與撰寫。
